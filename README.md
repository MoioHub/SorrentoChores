# Sorrento Chores

App web mobile-first per tracciare le faccende domestiche e la lista della spesa di casa.

Stack: React + Vite + Tailwind, database/storage/realtime via Supabase, deploy su GitHub Pages tramite GitHub Actions.

## Sviluppo locale

1. `npm install`
2. Copia `.env.example` in `.env` e compila `VITE_SUPABASE_URL` e `VITE_SUPABASE_ANON_KEY`
3. `npm run dev`

## Messa in produzione

Vedi il file `HANDOFF.md` per la procedura passo passo.
